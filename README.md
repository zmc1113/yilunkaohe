                                                                费马大定理介绍
费马大定理，又被称为“费马最后的定理”，由17世纪法国数学家皮耶·德·费玛提出。 它断言当整数n >2时，关于x, y, z的方程 x^n + y^n = z^n 没有正整数解。 德国佛尔夫斯克曾宣布以10万马克作为奖金奖给在他逝世后一百年内，第一个证明该定理的人，吸引了不少人尝试并递交他们的“证明”。 被提出后，经历多人猜想辩证，历经三百多年的历史，最终在1995年被英国数学家安德鲁·怀尔斯彻底证明。
